## 审查维度和权重

### 1. 现代 C++ 实践（25分）
- 使用智能指针（`std::unique_ptr`, `std::shared_ptr`）替代原始指针
- 避免使用 `new`/`delete`，优先使用 RAII
- 使用 STL 容器（`std::vector`, `std::map` 等）替代 C 风格数组
- 使用 `auto` 关键字提高可读性
- 使用移动语义（`std::move`）避免不必要的拷贝
- 使用 `nullptr` 替代 `NULL`
- 使用 `override` 关键字标记重写函数
- 使用范围 for 循环
- 使用 lambda 表达式

### 2. 内存安全（30分）
- 内存泄漏风险（未配对的 new/delete）
- 悬空指针和野指针
- 缓冲区溢出（数组越界）
- 使用已释放的内存
- 未初始化的变量
- 智能指针的正确使用
- 避免返回局部变量/临时对象的引用

### 3. 性能优化（15分）
- 不必要的对象拷贝
- 算法复杂度分析
- 避免过早优化
- 正确使用 `const` 引用传递大对象
- 容器预留空间（`reserve`）
- 避免在循环中重复分配内存
- 使用 `emplace_back` 替代 `push_back`

### 4. 异常安全（10分）
- 异常规范（`noexcept`, `throw()`）
- RAII 惯用法确保资源正确释放
- 避免在析构函数中抛出异常
- 使用智能指针自动管理资源

### 5. 线程安全（10分）
- 数据竞争检测
- 死锁风险
- 互斥锁的正确使用
- 原子操作的使用
- 避免在多线程环境下使用不安全的 STL

### 6. QT 最佳实践（10分，如果是 QT 项目）
- 正确使用信号槽机制
- `Q_OBJECT` 宏的正确使用
- QT 的父子对象内存管理
- QT 容器的使用（`QVector`, `QMap` 等）
- 元对象系统的正确使用
